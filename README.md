# MyAccess Launch Campaign

Welcome to the MyAccess marketing campaign documentation.

**MyAccess** is a disabled user testing service run by [MyVision Oxfordshire](https://www.myvision.org.uk/), a visual impairment charity. This campaign targets UK web agencies to introduce the service and drive pilot signups.
