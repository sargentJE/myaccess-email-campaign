## How to Use

1. **Each subfolder contains:**
   - `post.md` – The post copy, image reference, alt text, and notes
   - The relevant image file (where applicable)

2. **Before posting:**
   - Replace any `{{placeholders}}` with actual values (dates, times, links)
   - Add the registration/overview link in comments (not in main post)
   - Write appropriate alt text when uploading the image

3. **Posting tips:**
   - Post from Mark's personal profile for best reach
   - Cross-post or adapt for MyVision organisation page
   - Best times: Tuesday–Thursday, 8–10am or 12–2pm UK time

---

## Placeholders to Replace

- `{{webinar_date}}` – The actual webinar date
- `{{webinar_time}}` – The actual webinar time (e.g., "10:00am GMT")
- `[link]` or `[Registration link in comments]` – Your actual URLs
